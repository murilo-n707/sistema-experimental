# sistema-experimental
Um primeiro projeto em java de um sistema bancário.


Este é um projeto de simulação bancária desenvolvido em Java, focado em lógica de programação e manipulação de dados. O sistema permite operações básicas como depósitos, saque e consultas de saldo.

## Funcionalidades

* Gerenciamento de conta bancária;
* Validação de operações (Saques/Depósitos);
* Interface via linha de comando.

## Como Executar

O projeto já contém um executável `.jar` pronto. Para facilitar, utilize os scripts de inicialização.

### No Windows:
1. Localize o arquivo `abrir_programa.bat`.
2. Dê um clique duplo para iniciar.

### No Linux/macOS:
1. Abra o terminal na pasta do projeto.
2. Dê permissão de execução: `chmod +x abrir_banco.sh`
3. Execute o script: `./abrir_programa.sh`

---

## Tecnologias Utilizadas

* **Java**: Linguagem na qual foi escrito o programa;
* **Batch (Windows)**: Script de automação `.bat`;
* **Bash (Linux/Mac)**: Script de automação `.sh`;
* **Git**: Controle de versionamento.

## Estrutura do Repositório

* `untitled1.jar`: Executável do sistema.
* `abrir_banco.bat`: Launcher para Windows.
* `abrir_banco.sh`: Launcher para Linux/macOS.
